# 基本类型

ANSI标准规定一个字符为1个字节或多个字节，  
UNICODE为宽字节字符串或者UNICODE字符串

| bit | 位 | 基本单位 |  |
| --- | --- | --- | --- |
| byte | 字节 | 8位 | BYTE=unsigned char |
| word | 字 | 16位 | WORD=unsigned short |
| dword | 双字 | 32位 | DWORD=unsigned long |

## 字长

一次最大并行处理的二进制位数，CPU主要技术指标

## 原码、反码、补码

计算机系统中



1. 为了表示负数，最高位为符号位
2. 正数的原码、反码、补码均相同
3. 反码=原码符号位不变其它按位取反
4. 补码=反码+1
